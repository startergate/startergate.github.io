# [STARTERGATE.DEV](https://startergate.dev)

Built with Gatsby.js, Deployed by GitHub Pages

## Prerequisites

- Node.js
- npm

## Getting Started

```bash
npm install
```

## Scripts

| Command | Description |
|---|---|
| `npm start` | Start development server at `localhost:8000` |
| `npm run build` | Build for production |
| `npm run serve` | Serve the production build locally |
| `npm run clean` | Clear Gatsby cache and public directory |
| `npm run format` | Format source files with Prettier |
| `npm run deploy` | Build and deploy to GitHub Pages (manual) |

On every new GitHub Release, GitHub Actions automatically runs `format` → `build` → deploy. Manual runs are also available from the Actions tab.
